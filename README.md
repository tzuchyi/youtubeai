## 介紹
分析YouTube發佈的影片底下留言，將資料標記分為五大指標，針對`影片的喜愛`、 `YouTuber的喜愛`、`激動`、`諷刺`、`腥羶色`進行程度標記，針對五個標準的評分使用`FastText`模型進行訓練與預測。

每部影片的分析結果以網站和LINE Bot呈現，使用者只要輸入影片網址，就能即時爬蟲追蹤留言，進行YouTube影片的輿情分析，提供視覺化呈現，將非結構化訊息整理為有系統的資料。
